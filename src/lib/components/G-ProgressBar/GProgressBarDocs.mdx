## Usage

```svelte
<ProgressBar 
  progressAmt={65} 
  color="#733635"
  precision={0}
  tweenDuration={400}
  animate={true}
  labelInside={true}
  labelTextOutside="Progress"
/>
```

As a Web Component:

```html
<garnet-progressbar 
  progressAmt="65" 
  color="#733635"
  labelInside="true"
  labelTextOutside="Progress">
</garnet-progressbar>
```

## Behavior

The ProgressBar component:
- Animates smoothly between progress values when `animate` is true
- Provides visual feedback through a color-filled bar that grows according to the progress percentage
- Displays the progress percentage, optionally with decimal precision
- Supports two different layout modes: label inside or outside the bar
- Implements ARIA attributes for accessibility

## Design Considerations

This component follows Garnet UI's visual style, with the following characteristics:

- Rounded corners for a modern appearance
- Smooth transitions to enhance user experience
- Flexible color customization
- Accessible design with proper contrast ratios
- Responsive layout that adapts to different container sizes
- Support for both animated and immediate updates
