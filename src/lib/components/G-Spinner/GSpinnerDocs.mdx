# Spinner

The Spinner is a loading/progress component that indicates to the user that an action is in process. It has three animation variants available.

- [Circle](#circle) - Circular rotation animation
- [Jumper](#jumper) - Bounce effect animation
- [Jellyfish](#jellyfish) - Undulating jellyfish-like effect
- [Paused Animation](#paused-animation) - Example of paused animation

## Usage

```svelte
<script>
  import Spinner from "./Spinner.svelte";
</script>

<!-- Circle Spinner -->
<Spinner variant="circle" color="#733635" size="40" />

<!-- Jumper Spinner -->
<Spinner variant="jumper" color="#733635" duration="1s" size="60" />

<!-- Jellyfish Spinner -->
<Spinner variant="jellyfish" color="#733635" duration="2.5s" size="60" />

<!-- Paused Animation -->
<Spinner variant="jellyfish" color="#733635" duration="2.5s" size="60" pause={true} />
```
