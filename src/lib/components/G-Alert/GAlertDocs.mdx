## Usage

```svelte
<Alert 
  show={true}
  type="info"
  title="Information Title"
  description="This is an important message"
  showIcon={true}
  showAnimation={true}
/>
```

### The Alert component is particularly useful in:

* Form validation feedback to display errors or success messages
* System notifications about important events or status changes
* Confirmation messages after user actions (save, delete, update)
* Warning users about potential issues or consequences
* Drawing attention to critical information that requires immediate awareness
* Providing contextual help or tips in different states of an application
* Displaying temporary status messages that automatically dismiss after viewing

## Design Considerations

This component follows Garnet UI's visual style, with the following characteristics

* Color-coded visual system (blue for info, yellow for warnings, red for errors, green for success)
* Consistent spacing between icon and text elements for improved readability
* Animated appearance to draw appropriate attention without being jarring
* Clear visual hierarchy between title and description content
* Appropriate contrast ratios to ensure accessibility compliance
* Compact presentation to minimize disruption to the main content
* Responsive design that maintains legibility across device sizes
* Intuitive dismissal mechanism through the close button
* Visual reinforcement through carefully selected iconography
