The Tooltip component provides contextual information or additional guidance when users hover or focus on an interactive element.

## Usage

```svelte
<Tooltip 
  tip="This is helpful information"
  label="Additional details"
  position="bottom"
  showHTML={false}
  timeout="400"
/>
```

## Behavior

The Tooltip component appears when users hover or focus on the trigger button. It includes a configurable delay for both showing and hiding to prevent accidental triggering. The tooltip can be dismissed by pressing Escape, clicking elsewhere, or moving focus away from the trigger.

## Accessibility

This component follows WCAG accessibility guidelines with:

- Full keyboard navigation support
- Proper ARIA attributes (aria-expanded, aria-describedby)
- Support for screen readers
- Respect for user motion preferences

## Design Considerations

This component follows Garnet UI's visual style, with the following characteristics:

- Subtle elevation and shadow effects
- Smooth transitions to enhance user experience
- Responsive design that adapts to different screen sizes
- Directional indicators that point to the trigger element
- Support for dark mode through prefers-color-scheme
